# Prompt-engineering-for-agentic-ai-systems-with-LangChain-and-OpenAI

# Project title:
Operational prompt engineering for NLP and LLM supported systems

# Key tools: 
•	Jupyter Notebook | dotenv | LangChain  | OpenAI api | prompt engineering strategies | DuckDuckGo search api 

# Project objective:
Key objectives are: 
•	Apply prompt engineering mechanisms
•	Optimize AI created results with the help of LLM  
•	Create a basic framework supported for Agentic AI systems
•	Present different prompting approaches such as zero-shot prompting, few-shot prompting, chain of thought reasoning, python code generation, DuckDuckGo 
applied for web search, LLM role guided prompting, structured prompt for LLM
